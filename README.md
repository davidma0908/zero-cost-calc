# 零成本持倉計算機

> 輸入持倉資訊，立即算出需賣出多少部位可回本，剩餘股份即為零成本持倉。

🌐 **線上使用**：[davidma0908.github.io/zero-cost-calc](https://davidma0908.github.io/zero-cost-calc)

---

## 什麼是零成本持倉？

買入股票後，透過部分賣出回收原始投入本金，剩餘持股的成本即歸零。
即使股價之後下跌，也不會有實際虧損——剩下的全是獲利。

---

## 使用方式

### 1. 輸入持倉資訊

| 欄位 | 說明 |
|------|------|
| **持有股數** | 目前帳上的總持倉數量（股） |
| **平均成本** | 所有加倉後的加權平均買入價（元/股） |

### 2. 輸入當前價格

提供兩種輸入模式，可切換：

- **獲利%**：輸入目前浮盈百分比（如 `+30`），也可拖曳滑桿快速調整
- **股價**：直接輸入當前市價

### 3. 選擇回收目標

點選下方籌碼快速切換：

| 目標 | 說明 |
|------|------|
| 回收 25% | 賣出部位取回 1/4 本金 |
| 回收 50% | 賣出部位取回 1/2 本金 |
| **回收 100%** | 完全零成本（預設） |
| 回收 125% | 回本並多賺 25% |
| 回收 150% | 回本並多賺 50% |

### 4. 查看結果

| 結果 | 說明 |
|------|------|
| **需賣出（圓弧）** | 達成目標需賣出的股數 |
| **賣出回收金額** | 賣出後取回的現金 |
| **零成本剩餘持倉** | 回本後仍持有的股數（全為獲利） |
| **當前股價** | 計算用的股價（紅/綠代表虧損/獲利） |
| **總投入成本** | 持有股數 × 平均成本 |

---

## 計算公式

```
當前股價     = 平均成本 × (1 + 獲利% / 100)
需回收金額   = 總投入成本 × (回收目標% / 100)
需賣出股數   = ⌈ 需回收金額 / 當前股價 ⌉
零成本持倉   = 持有股數 − 需賣出股數
```

---

## 範例

| 條件 | 數值 |
|------|------|
| 持有股數 | 10,000 股 |
| 平均成本 | $50 元 |
| 當前獲利 | +30% → 股價 $65 |
| 回收目標 | 100% |

```
總投入     = 10,000 × $50   = $500,000
需賣出     = ⌈500,000 / 65⌉ = 7,693 股
零成本持倉 = 10,000 - 7,693  = 2,307 股（市值 $150,000，全為純獲利）
```
